# ATM Management System using Flask

## Project Overview

This is a simple **ATM Management System** developed using **Python Flask** with a frontend and backend. The application allows users to create accounts, deposit money, withdraw money, view account details, and delete accounts.

The project demonstrates the fundamentals of:

* Flask Web Framework
* Frontend & Backend Integration
* Form Handling
* Routing in Flask
* Basic Banking Operations
* In-Memory Data Storage using Python Dictionary

---

##  Features

### 1. Create Account

* Create a new bank account.
* Enter Account Number, Account Holder Name, and Initial Balance.

### 2. Dashboard

* View account details.
* Display account holder name and current balance.

### 3. Deposit Money

* Add money to the account balance.

### 4. Withdraw Money

* Withdraw money from the account.
* Checks for sufficient balance before withdrawal.

### 5. Delete Account

* Remove an existing account from the system.

##  Installation & Setup

### Step 1: Clone Repository

```bash
git clone https://github.com/shaik-tanveer-dev/atm-management-system-flask.git
```

### Step 2: Navigate to Project Folder

```bash
cd ATM-Flask-Project
```

### Step 3: Install Flask

```bash
pip install flask
```

### Step 4: Run Application

```bash
python app.py
```

### Step 5: Open Browser

```text
http://127.0.0.1:5000
```
